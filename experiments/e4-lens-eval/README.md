# E4 — Quantifying lens false positives (C5, P1)

Run the six official evaluation sets in `data/evaluations/lens-eval-*.json`
(association / multihop / multilingual / order-ops / poetry / typo) and report
per-set false-positive and miss rates.

The paper itself calls the lens "undoubtedly an imperfect method"; this
experiment puts numbers on *how* imperfect. If time-boxed, run 2–3 sets.
